# Ansgar Harmeier — Portfolio

A single-page personal portfolio for Ansgar Harmeier, a HubSpot CRM, Automation, AI & Agents consultant based in Düsseldorf, Germany.

No build step — it's a self-contained static site with zero runtime dependencies. Everything lives in `index.html`; the only thing fetched at runtime is Google Fonts.

## Features

- **Sections** — Hero, About, Skills, Experience, Education, Interests, Contact.
- **Light / dark theme** — driven by the `data-theme` attribute on `<html>`.
- **Font switching** — three typeface sets (`grotesk`, `serif`, `plex`) via `data-font`.
- **Bilingual (EN / DE)** — text swaps through `data-i18n` / `data-i18n-html` attributes and a language toggle.
- **Hero portrait** — a real photo (`portrait.jpg`) rendered with `object-fit: cover`.

## Files

| File | Purpose |
|------|---------|
| `index.html` | The full page — markup, design tokens, styles, i18n strings, and behavior. |
| `portrait.jpg` | Hero portrait photo. |
| `favicon.svg` | AH monogram favicon. |
| `apple-touch-icon.png` | 180×180 home-screen icon for iOS. |

The site was originally generated by Claude's design runtime; the design-runtime scaffolding (a fillable `<image-slot>` web component and a React/Babel "Tweaks" panel) has since been removed, leaving a dependency-free static page.

## Run locally

```bash
python3 -m http.server 8765
# then open http://localhost:8765/
```

(Opening `index.html` directly via `file://` also works now that there's no runtime JSX fetch.)

## Dependencies

None to install. The only runtime fetch is **Google Fonts** (Schibsted Grotesk, Libre Franklin, Newsreader, Space Grotesk, IBM Plex Sans/Mono).

## Deployment

Hosted on Vercel (static, no build step) and live at **https://ansgarharmeier.vercel.app**. Deploy with the Vercel CLI from the project root:

```bash
vercel --prod
```

Source is on GitHub at https://github.com/Aharmeier/ansgarharmeier.
